# Road Eye India
Edge AI dashcam on Raspberry Pi for road anomaly detection (potholes, obstacles).  
Model: YOLOv8 TFLite INT8.  
Runs headless or with display; saves full video, event clips, snapshots, optional audio.

## Paths
- Code: `src/`
- Models: `models/`
- Data outputs: `full/`, `clips/`, `snapshots/`

## Run


Press `q` to exit display window.
